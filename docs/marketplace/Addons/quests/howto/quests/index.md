Let's start with the basics.

Quest Books are given to each player when they spawn. However, if you lose yours, you can craft a new one using a regular book.

To open the Quest Book, hold it in your main hand and press the "Use" button (right-click on PC, left trigger on consoles).

On the main menu you'll see buttons for Create Quests (Creative only), Pinned Quests (if you have pinned any quests), Timed Quests and Quest Chapters.

After choosing a chapter, you'll see all the quests within it. Clicking on a quest will show its requirements and its rewards.

Note: Settings in the Guide Book are Operator only.
